# 测试数据准备方案 — 宠物行为视频素材

| 项目 | 内容 |
|------|------|
| **文档编号** | TDP-2026-001 |
| **关联变更** | CHG-2026-001 |
| **目的** | 为AI行为识别模块测试准备视频数据集 |
| **编制日期** | 2026-06-09 |

---

## 一、素材需求
### 行为覆盖
进食、饮水、运动/跑动、睡眠/休息、排泄、异常行为

### 品种覆盖
猫(小-中)、小型犬、中型犬、大型犬

## 二、视频来源
- YouTube宠物24h直播 (10+频道)
- B站萌宠直播 (5+频道)
- 公开数据集: Animal Pose, PetTrack, DogCentric Activity
- 自录: 宠物主人/宠物店/医院录制

## 三、预处理标准
| 参数 | 标准 |
|------|------|
| 分辨率 | 640x480 或 1280x720 |
| 帧率 | 25fps |
| 编码 | H.264 MP4 |
| 时长 | 5min-60min/段 |

## 四、24h模拟方案
00:00-06:00睡眠 → 06:00-08:00早晨 → 08:00-12:00日间 → 12:00-14:00午间进食 → 14:00-18:00下午 → 18:00-20:00晚间 → 20:00-24:00夜间

## 五、快速启动测试集
cat_day(30min), cat_eat(10min), dog_day(30min), dog_sleep(20min), multi_pet(15min)
